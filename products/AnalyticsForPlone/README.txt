About

	Add Google Analytics in your Plone!

Dependencies

	Plone 2.1
	Zope 2.8

Install

	1. Copy the directory of this product in your product's directory of your Zope instance.

	2. Go to the Quick Installer and install Analytics for Plone.

	3. Go to the Analytics for Plone Configlet and set up your Google's id.

	4. That's all.

Download

	* On Plone.org : http://plone.org/products/analyticsforplone

	* On SVN Collective : http://svn.plone.org/svn/collective/AnalyticsForPlone
